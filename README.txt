
ABOUT
-----
  This module provides a basic API interface for Drupal modules to use the Dun
  & Bradstreet API Endpoint. This is not an official module and is only intended
  as a useful supplementary for programming your own functions.


REQUIREMENTS
------------
  1) You need a DNB secret, obtained from the Direct+ API program.

  2) This requires the Guzzle Library to be installed in your Drupal site.
     https://github.com/guzzle/guzzle
